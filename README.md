# Climatic
Climatic is a weather app where you can enter the name of any city in the world and get real-time weather information. I built it by following the workshop instructions by @gautamjajoo, while adding my own creative twist. I included different backgrounds and icons that change depending on whether the weather is cloudy, rainy, or clear

I used HTML for the structure, CSS for the design, and JavaScript to connect the API and dynamically change the backgrounds and icons

This was my first project working with APIs, and I’m really proud of how it turned out. I hope you like it!
